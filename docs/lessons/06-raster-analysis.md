# Lesson 06 · Raster Analysis

## Learning goals

- Explain what raster cells represent.
- Inspect raster values.
- Symbolize raster data.
- Reclassify raster values.
- Combine raster layers for simple suitability analysis.

## Key concepts

Raster data stores information in a grid of cells. Each cell has a value. Raster analysis is especially useful for elevation, land cover, temperature, precipitation, distance surfaces, and suitability modeling.

## Workflow

1. Add a raster layer.
2. Inspect cell values.
3. Adjust raster symbology.
4. Reclassify values.
5. Combine criteria for a suitability map.

## Practice task

Create a simple suitability layer using at least two raster criteria.

## Deliverable

Submit the final raster map and a short explanation of your suitability logic.

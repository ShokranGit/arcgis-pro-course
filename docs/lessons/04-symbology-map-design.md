# Lesson 04 · Symbology and Map Design

## Learning goals

- Change layer symbols.
- Use graduated colors or graduated symbols.
- Choose classification methods.
- Add labels and improve readability.
- Make design choices that match the data.

## Key concepts

Map design is argument design. Your choices about color, classification, labels, and scale influence what readers notice.

## Workflow

1. Open the Symbology pane.
2. Choose a symbology method.
3. Adjust color ramps and class breaks.
4. Add labels if appropriate.
5. Check readability at the intended scale.

## Practice task

Create two versions of the same map using two different classification methods. Compare how the visual pattern changes.

## Deliverable

Submit both maps and a short explanation of which one communicates the pattern more clearly.

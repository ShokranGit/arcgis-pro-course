## Learning goals

- Distinguish between vector and raster data.
- Recognize common GIS file formats.
- Explain why coordinate systems matter.
- Check a layer's spatial reference in ArcGIS Pro.

## Key concepts

GIS layers are visual representations of spatial data. A layer can point to data stored in a shapefile, geodatabase feature class, raster file, web service, or other source.

Coordinate systems allow GIS software to place spatial data in the correct location on the earth. If layers do not line up, projection or spatial reference issues are often the first thing to check.

## Workflow

1. Add multiple layers to a map.
2. Open layer properties.
3. Check the source and spatial reference.
4. Compare vector and raster examples.
5. Change the map coordinate system if needed.

## Practice task

Add two vector layers and one raster layer. Identify the data type and coordinate system of each.

## Deliverable

Submit a short table listing each layer, data type, file/source type, and coordinate system.

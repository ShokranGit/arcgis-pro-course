## Learning goals

By the end of this lesson, you should be able to:

- Open ArcGIS Pro and create a new project.
- Identify the ribbon, map view, Catalog pane, Contents pane, and Geoprocessing pane.
- Add a layer to a map.
- Save a project and understand where project files are stored.

## Key concepts

ArcGIS Pro projects organize maps, layouts, toolboxes, databases, and connections. The project is not just a single map; it is a workspace.

## Workflow

1. Open ArcGIS Pro.
2. Create a new project.
3. Choose a clear project name and folder location.
4. Open the Catalog pane.
5. Add or connect to a folder.
6. Add a sample layer to the map.
7. Save the project.

## Practice task

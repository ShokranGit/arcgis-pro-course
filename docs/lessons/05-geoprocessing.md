# Lesson 05 · Geoprocessing

## Learning goals

- Open and use the Geoprocessing pane.
- Run common tools such as Buffer, Clip, Dissolve, and Intersect.
- Understand inputs, outputs, parameters, and environments.
- Save outputs in a clear location.

## Key concepts

Geoprocessing tools transform spatial data. Each tool has inputs, parameters, and outputs. A good GIS workflow is both analytical and organized.

## Common tools

| Tools | Uses |
|---|---|
| Buffer | Create zones around features |
| Clip | Cut one layer by the boundary of another |
| Dissolve | Merge features by shared attributes |
| Intersect | Find overlapping areas between layers |
| Spatial Join | Transfer attributes based on location |

## Practice task

Run at least two geoprocessing tools and document the input layer, parameters, and output layer for each.

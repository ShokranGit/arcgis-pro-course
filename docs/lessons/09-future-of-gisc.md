# Lesson 09 · The Future of GISc

## Learning goals

- Identify emerging trends in GIS and geospatial technology.
- Discuss how real-time data, cloud computing, and AI are changing the GIS field.
- Reflect on how these trends might shape future GIS careers, including your own.
## Key concepts

GIS is increasingly real-time and connected: sensors, GPS-enabled devices, and IoT feeds now stream location data continuously instead of arriving as a single static dataset. Cloud-based GIS platforms let organizations publish and share maps and analysis without every user needing desktop software, and machine learning is starting to assist with tasks like feature extraction from imagery and predictive spatial modeling.

These shifts are also making GIS more accessible to non-specialists through web maps and dashboards, which raises the same questions about data quality, bias, and interpretation introduced in Lesson 00 — they just apply at greater speed and scale.

## Workflow

1. Read the assigned material on emerging trends in mapping technology.
2. Identify one emerging trend (real-time GIS, cloud GIS, AI-assisted mapping, and so on) that interests you.
3. Research a real organization or product that uses that trend.
4. Connect the trend back to a concept or skill from earlier in the course.

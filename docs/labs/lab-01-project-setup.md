# Lab 01 · Project Setup and Interface

## Purpose

This lab makes sure every student can create an ArcGIS Pro project, connect to course data, add layers, and navigate the interface.

## Required files

- Course starter data folder
- One vector layer
- One basemap

## Instructions

1. Create a new ArcGIS Pro project.
2. Save it inside your course folder.
3. Connect to the course data folder.
4. Add one vector layer to the map.
5. Open the attribute table.
6. Take a screenshot showing the map, Contents pane, Catalog pane, and attribute table.

## Deliverables

- Screenshot of your ArcGIS Pro workspace.
- Short paragraph explaining where your project and data are saved.

## Grading checklist

- [ ] Project folder is clearly named.
- [ ] Course data folder is connected.
- [ ] At least one layer is visible.
- [ ] Screenshot includes required interface elements.
- [ ] Written explanation is clear.

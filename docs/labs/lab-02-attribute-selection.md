# Lab 02 · Attribute Selection

## Purpose

This lab practices selecting features using attribute queries and exporting selected features as a new layer.

## Required files

- A feature layer with an attribute table
- ArcGIS Pro project from Lab 01

## Instructions

1. Open the project from Lab 01.
2. Add the assigned feature layer.
3. Open the attribute table.
4. Identify two useful fields for selection.
5. Use **Select By Attributes** to build a query.
6. Apply the selection.
7. Export the selected features as a new layer.
8. Symbolize the exported layer so it is visually distinct.

## Deliverables

- Screenshot of your query.
- Screenshot of selected features on the map.
- Exported layer name.
- Short explanation of what your query selected and why.

## Grading checklist

- [ ] Query is valid.
- [ ] Selected features match the stated logic.
- [ ] Exported layer is saved clearly.
- [ ] Screenshots document the process.
- [ ] Explanation connects the query to the spatial question.

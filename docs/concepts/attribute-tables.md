# Attribute Tables

Attribute tables store information about features.

Each row usually represents one feature. Each column is a field.

## Why attribute tables matter

Attribute tables let you:

- Inspect data values
- Select features
- Calculate new fields
- Join tables
- Symbolize maps
- Summarize patterns

## Common field types

| Type | Example |
|---|---|
| Text | Borough name |
| Integer | Population count |
| Double/Float | Rate, percentage, area |
| Date | Inspection date |

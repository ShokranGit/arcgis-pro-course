# Vector and Raster Data

GIS data often comes in two major forms: vector and raster.

## Vector data

Vector data represents features using points, lines, and polygons.

| Geometries | Examples |
|---|---|
| Point | Trees, schools, subway stations |
| Line | Roads, streams, transit routes |
| Polygon | Parcels, neighborhoods, flood zones |

## Raster data

Raster data represents space as a grid of cells. Each cell stores a value.

Common raster examples include elevation, temperature, land cover, satellite imagery, and suitability scores.

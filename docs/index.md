# ArcGIS Pro Course

A practical learning hub for building GIS skills with ArcGIS Pro. This site organizes tutorials, labs, datasets, troubleshooting notes, and core GIS concepts so students can move from software basics to independent spatial analysis.

## Course pathway

The course is organized as a sequence of modules and labs. Each lab introduces a focused set of GIS concepts and ArcGIS Pro skills.

<div class="card-grid" markdown>

<div class="course-card" markdown>
<span class="tag">Module</span>

### Introduction, Foundations, and Data Management
Course overview, software setup, GIS/GISc applications, map types, ArcGIS interface, data types, file management, data acquisition, adding data, geodatabases, and coordinate systems.
</div>

<div class="course-card" markdown>
<span class="tag">Module</span>

### Tabular Data and Mapping
Attribute tables, CSV table creation, field editing and calculation, cartographic principles, map classification, layout design, and data selection.
</div>

<div class="course-card" markdown>
<span class="tag">Module</span>

### Spatial Analysis
Table joins, spatial joins, intersect relationships, and distance-based spatial relationships.
</div>

<div class="course-card" markdown>
<span class="tag">Module</span>

### Geoprocessing Tools
Buffer, clip, dissolve, overlay, union/intersect, raster analysis, and introductory geospatial programming.
</div>

<div class="course-card" markdown>
<span class="tag">Module</span>

### Ethics & Future
Map misuse, ethical issues in GISc, and the future of GIS.
</div>

</div>
